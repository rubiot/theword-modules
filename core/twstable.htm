theWord stable version

Note! If you have never downloaded theWord Stable version from Add Titles you will need at least this file to get started. Download this and replace your theword.exe with it.

Note this will only work correctly if start from "theword.exe" not a different name and you must have the file above first!


Before installing it is recommended that you make a backup of theWord and your files for theWord.


theWord Stable is the beta version of theWord where things are nearing release version.
