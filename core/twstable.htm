theWord stable version

Important:

1. Before installing it is recommended that you make a backup of theWord and your files for theWord.

2. This will only work on a compact install or normal install that is not in Program Files or Program Files (86x).

3. If you have never downloaded theWord Stable version from Add Titles you will need at least this file to get started. Download this and replace your theword.exe with it.


theWord Stable is the beta version of theWord where things are nearing release version.
